title Single page app (SPA)

browser->server: User posts new note

note over server:
The browser only sends 1 request
The POST request to the 
address new_note_spa
contains the new note as JSON-data
end note 

note over server: 
The Content-Type header of the request
tells the server that the included data
is represented in the JSON format
end note 

server-->browser:The server responds with status code 201 created

note over browser:
The server does not ask for a redirect,
the browser stays on the same page,
and it sends no further HTTP requests
end note 

note over browser:
Then the event handler creates a new note
adds it to the notes list
with the command notes.push(note)
rerenders the note list on the page
and sends the new note to the server.
end note
